# Code Alchemist Installer

npx 安装器 for Code Alchemist

## 使用方法

```bash
# 安装到当前项目
npx code-alchemist

# 更新
npx code-alchemist --update

# 帮助
npx code-alchemist --help
```

## 发布到 npm

```bash
cd installer
npm login
npm publish
```
